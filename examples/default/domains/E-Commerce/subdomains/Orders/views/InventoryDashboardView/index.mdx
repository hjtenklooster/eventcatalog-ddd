## Overview

The Inventory Dashboard View aggregates inventory adjustment events into a real-time projection of stock levels across all warehouses, enabling warehouse managers to make informed restocking decisions.

## Data Shape

| Field | Source Event | Description |
|-------|-------------|-------------|
| productId | InventoryAdjusted | Product identifier |
| warehouseId | InventoryAdjusted | Warehouse location |
| currentStock | InventoryAdjusted | Computed current stock level |
| lastAdjusted | InventoryAdjusted | Last adjustment timestamp |
