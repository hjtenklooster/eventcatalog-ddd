## Overview

The Order Summary View is a read model that combines order creation and status update events into a denormalized projection optimized for customer support workflows.

## Data Shape

| Field | Source Event | Description |
|-------|-------------|-------------|
| orderId | OrderCreated | Unique order identifier |
| status | OrderStatusUpdated | Current order status |
| customerName | OrderCreated | Customer display name |
| totalAmount | OrderCreated | Order total |
| lastUpdated | OrderStatusUpdated | Last status change timestamp |
