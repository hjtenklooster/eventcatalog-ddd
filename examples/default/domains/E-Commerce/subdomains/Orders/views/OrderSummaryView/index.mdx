## Overview

The Order Summary View is a read model that combines order confirmation and amendment events into a denormalized projection optimized for customer support workflows.

## Data Shape

| Field | Source Event | Description |
|-------|-------------|-------------|
| orderId | OrderConfirmed | Unique order identifier |
| status | OrderAmended | Current order status |
| customerName | OrderConfirmed | Customer display name |
| totalAmount | OrderConfirmed | Order total |
| lastUpdated | OrderAmended | Last amendment timestamp |
